# Домашнее задание 5. Анализ данных на Spark SQL

Проект для работы с Apache Spark SQL и выполнения запросов на выборку и агрегацию данных о бронированиях отеля.

## Структура проекта

### Основные файлы

- **`hw5.ipynb`** - Jupyter ноутбук с основным кодом

### Файлы с данными

Все данные находятся в директории `data/`:

- **`data/hotels.csv`** - данные о бронированиях отеля
  - Содержит информацию о бронированиях: ID, количество гостей, тип номера, даты, статус и т.д.

### Особенности работы с данными

- Данные содержат некорректную дату - 29 февраля 2018 года. Такой даты не существует, поэтому при такая
дата будет заменена на 28 февраля 2018 года.
